Wer sich eine eigene Website erstellen möchte wird häufig mit Fachbegriffen und Abkürzungen konfrontiert. Im Bereich der Website-Erstellung taucht oft der Begriff HTML auf.
Doch was ist HTML eigentlich und was hat das mit einer Website zu tun?

HTML bedeutet HyperText Markup Language und ist eine Auszeichnungssprache für Websites. Das Grundgerüst einer Website wird mit HTML beschrieben und dann von einem Webbrowser wie Firefox interpretiert und angezeigt.
